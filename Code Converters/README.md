Code converters project:
This file contains:
1) Binary to Gray Code converter in dataflow style(1) and structural style(2)
     Gray code is a unit distance code where only one bit changes at a time
   
2) Gray to Binary Code converter in dataflow style(1) and structural style(2)
     The inverse of the above, where the binary bits are recovered from Gray code
   
3) Binary to One Hot converter
     Converts a binary input into a format where only a single bit in the output vector is 'high' (1) at any given time.
   
4) BCD to Excess-3 Code Converter
     Converts Binary Coded Decimal to Excess-3 by adding 3 to the BCD value.
5) Basic Testbenchs and Self-Checking Testbenches to confirm the functionality of the code converters
